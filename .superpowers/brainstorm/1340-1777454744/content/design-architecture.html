DB migration: ALTER TABLE exams ADD COLUMN subject TEXT — backward safe (nullable, existing exams get null subject)

Component Tree

Data Flow

1. Exam created → subject stored in DB alongside name + date → fetchExam() loads it into context
2. selectedExam set → auto-selected (nearest exam) or manually switched → ExamCard reads selectedExam.subject
3. filteredWeakTopics → useMemo inside ExamCard: weakTopics.filter(t => normalizeSubject(t.subject) === normalizeSubject(selectedExam.subject)) → top 5 by count
4. "Practice" clicked → sessionStorage.setItem('amn_prefill', { subject, topic }) → router.push('/quiz')
