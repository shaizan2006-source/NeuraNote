Animation Stack for FocusModeCard

  • Ambient glow blob — radial-gradient cyan behind ring, Framer animate opacity [0.3→0.7→0.3] on 3s loop
  • Two pulsing rings — same pattern as CallTutorCard's mic orb, uses scale [1→1.2→1] + opacity [0.3→0→0.3]
  • 12 tick marks — SVG clock ticks via rotation transforms, subtle cyan at 25% opacity
  • Ring glow — filter: drop-shadow(0 0 6px #22d3ee) on the stroke arc
  • "Start focus" label — colored cyan instead of grey to draw eye
  • Hover — scale 1.04, y:-4, box-shadow: 0 0 40px rgba(34,211,238,0.25)
