3 · Scroll Motion System

Lerp Factor

0.06

Each rAF tick, current position moves 6% toward target. Creates physical lag — the glow feels weighted, not glued to scroll.

Range: 0.03 (dreamlike) → 0.12 (snappy)

Far Layer Parallax

3%

Scroll 1000px → bloom shifts 30px upward. Atmospheric. You feel the depth without seeing explicit movement.

translate3d — GPU only, zero paint

Mid Layer Parallax

7%

Slightly more reactive. Scroll 1000px → 70px shift. The difference between far (30px) and mid (70px) creates the depth illusion.

2× faster than far = parallax depth

rAF Optimization

idle-stop

The rAF loop only runs while scroll is in motion. When delta < 0.1px it cancels itself. Zero CPU while user isn't scrolling.

Passive scroll listener throughout

5 · Tunable Configuration
Constant Default Effect
LERP 0.06 Interpolation speed. Lower = dreamier, higher = snappier.
FAR_PARALLAX 0.03 Far bloom movement ratio. 0.03 = 30px per 1000px scroll.
MID_PARALLAX 0.07 Mid wash movement ratio. Higher than far creates depth.
FAR_BREATHE_DURATION 14s Bloom pulse cycle. Longer = more meditative.
MID_BREATHE_DURATION 10s Wash pulse cycle. Offset from far to avoid sync.
FAR_OPACITY 0.18 Peak bloom opacity. Keep under 0.25 for Void feel.
SCROLL_SELECTOR .amn-focus-scroll CSS selector for the scroll container to listen on.
