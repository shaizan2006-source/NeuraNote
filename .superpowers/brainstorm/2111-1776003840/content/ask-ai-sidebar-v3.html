What's new in v3

Collapsible sidebar

Double-chevron button in header. Collapsed = 56px icon rail with tooltips on hover. Expand button to restore. State persists in localStorage.

SVG icons (Lucide)

Clean, consistent, renders identically on every device. Grid = Dashboard, MessageSquare = Ask AI, Upload = Upload, FileText = Notes.

Vertical navigation

Natural top-to-bottom scan. Better touch targets. Scales if you add pages later.

Collapsed PDF indicator

Even when collapsed, a small cyan-bordered file icon at the bottom shows the active PDF. Tooltip reveals the filename.

Mobile: Sidebar is hidden entirely — hamburger icon in top-left opens it as a slide-over overlay with the same expanded layout. Same behavior as current app.
