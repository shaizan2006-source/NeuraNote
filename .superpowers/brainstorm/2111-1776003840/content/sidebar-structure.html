Key differences

Option A: Minimal

  • Always visible with labels
  • No learning curve
  • Takes ~220px width always
  • Simpler to build
  • Collapses to hamburger on mobile

Option B: Icon-rail

  • Collapsed = ~56px, more room for content
  • Expands on hover to show labels + PDFs
  • Feels more premium/modern
  • Slightly more complex interaction
  • Same hamburger on mobile
