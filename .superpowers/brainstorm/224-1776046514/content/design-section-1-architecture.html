Key decisions
Drawer is not a new feature It's a lightweight shell around the existing Ask AI API. No separate endpoint, no separate data model.
Conversation continuity via URL param "Open full chat →" navigates to /ask-ai?cid={id}. Ask AI page reads param on mount and loads the thread.
dashboard_mode persists Stored in both localStorage (instant read) and Supabase (cross-device). localStorage wins on page load, syncs on mount.
Sidebar state is local-only Collapse preference stays in localStorage — not worth a Supabase round trip for a UI toggle.
