Option — subtle shadow
Spec: box-shadow: 0 4px 12px rgba(139,92,246,0.15) → hover 0 6px 16px rgba(139,92,246,0.25)

Tradeoff: Slightly more depth than no shadow, but still way less glowy than current. A middle ground.
My recommendation
Go with the simplified version (gradient only, no shadow). It keeps the premium feel, it's cleaner for a study app, and it respects the Ambient Intelligence philosophy. The gradient is enough — you don't need the glow to make a button look good. The hover lift (translateY) provides tactile feedback without noise.

If you want a tiny bit of depth, the subtle shadow option is a compromise, but I think it's unnecessary. The gradient alone is premium.
