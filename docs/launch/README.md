## How to Use

### First Time (Week 13)
1. Read `SPRINT_04_PRELAUNCH_AND_BETA.md` completely
2. Use `LAUNCH_COPY_AND_ASSETS.md` to write all copy (Day 88)
3. Day-by-day follow Claude Code prompts in SPRINT_04

### During Launch (Weeks 15-17)
1. Read `SPRINT_05_PUBLIC_LAUNCH.md` weekly
2. Execute day-by-day Claude Code tasks
3. Use `LAUNCH_COPY_AND_ASSETS.md` for templates
4. Track metrics using `METRICS_SQL_QUERIES.md`
5. Create weekly report in `weekly-reports/week-NN.md`

### Scale Phase (Weeks 18-24)
1. Read `SPRINT_06_SCALE_AND_MEASURE.md` weekly
2. Execute week-by-week Claude Code tasks
3. Run 5 key metrics queries every Monday
4. Store weekly report in `weekly-reports/week-NN.md`
